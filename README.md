To use this project nothing more simple:
- test directly on stackblitz 
- clone this repo by doing:
  
```shell
$ git clone https://github.com/TawalMc/SearchbarWithSelectAndFilter.git
```
then install dependencies and run 
```shell
$ yarn 
$ yarn dev
```
